##

### Installation

- Just, Clone this repository -
  ```
  git clone --depth=1 https://github.com/vakaar78/zphisher-baby.git
  ```

- Now go to cloned directory and run `zphisher.sh` -
  ```
  $ cd zphisher
  $ bash zphisher.sh
  ```

- On first launch, It'll install the dependencies and that's it. ***Zphisher*** is installed.

### Run on Docker

- Docker Image Mirror:
  - **DockerHub** : 
    ```
    docker pull baby/zphisher
    ```
  - **GHCR** : 
    ```
    docker pull[ https://github.com/vakaar78/zphisher-baby.git
    ```

- By using the wrapper script [**run-docker.sh**](https://raw.githubusercontent.com/https://github.com/vakaar78/zphisher-baby.git)

  ```
  $ curl -LO https://raw.githubusercontent.com/https://github.com/vakaar78/zphisher-baby.git
  $ bash run-docker.sh
  ```
- Temporary Container

  ```
  docker run --rm -ti Baby_Doll/zphisher
  ```
  - Remember to mount the `auth` directory.
